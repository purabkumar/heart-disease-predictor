❤️ Heart Disease Prediction System

This project is a Machine Learning-based web application that predicts whether a person is at risk of heart disease based on medical attributes.

---

🚀 Live Demo

(After deployment, add your Streamlit link here)

---

📌 Features

* Predicts heart disease risk using Machine Learning
* Built with Random Forest Classifier
* Interactive web interface using Streamlit
* Real-time prediction

---

🧠 Machine Learning

* Algorithm: Random Forest Classifier
* Accuracy: ~85%
* Dataset: UCI Heart Disease Dataset

---

📊 Input Features

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Rest ECG
* Max Heart Rate
* Exercise Induced Angina
* Oldpeak
* Slope
* CA
* Thal
